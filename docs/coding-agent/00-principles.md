# Principles

- Fast + memory-efficient Go TUI agent with minimal dependencies.
- Standalone first: full local use without SaaS.
- YOLO local mode only (no permission popups).
- Remote control optional and live-toggleable.
- Local + remote inputs merge (no takeover).
- Model switching supported.
- Pluggable compaction algorithms.
- Extensible tools + hooks.
- Sessions resumable locally (JSONL).
- Remote control secure (dial-out).
- Single owner per agent; full output streamed; no queueing.
- Library must be composable: reuse agentic-weave in SaaS assistants, TUI, and other bots.
