# Contributing

Thanks for contributing to Agentic Weave.

## Quick Start
1) Fork and clone the repo.
2) Create a branch for your change.
3) Run tests: `go test ./...`
4) Open a PR with a clear description.

## Guidelines
- Keep changes focused and small.
- Update docs when public APIs change.
- Add tests when behavior changes.
- Prefer simple interfaces over heavy abstractions.

## Reporting Issues
- Include repro steps, expected behavior, and actual behavior.
- Add logs or stack traces when possible.
